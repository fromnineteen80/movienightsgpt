# Privacy Policy

MovieNightGPT is a personal, user-controlled tool.

- No user data is collected or stored by the application.
- All movie preferences and history are stored in the user's own GitHub Gist.
- The application does not use cookies or analytics.
- No data is shared with third parties.
- GitHub API access is used only to update files explicitly owned by the user.

This tool runs entirely client-side and through user-authorized GitHub actions.
